# 🎬 Global CineMind AI

An AI-powered Movie Recommendation System that provides personalized recommendations for both Hollywood and Bollywood movies using Machine Learning.

---

## 🚀 Features

- 🎥 Hollywood Movie Recommendations
- 🎥 Bollywood Movie Recommendations
- 🤖 Content-Based Filtering
- 📊 Interactive Streamlit Dashboard
- ⚡ Fast Recommendation Engine
- 🎨 Clean User Interface

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit

## ▶️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/GlobalCineMindAI.git
```

Move into the project folder:

```bash
cd GlobalCineMindAI
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run dashboard/app.py
```
