# 🎬 Global CineMind AI

An AI-powered Movie Recommendation System that provides personalized recommendations for both Hollywood and Bollywood movies using Machine Learning and Streamlit.

## 🚀 Features

- Hollywood Movie Recommendations
- Bollywood Movie Recommendations
- Content-Based Filtering
- Interactive Streamlit Dashboard
- Fast Recommendation Engine
- Clean User Interface

## 🛠️ Tech Stack

- Python
- Pandas
- Scikit-Learn
- Streamlit
- Machine Learning

## 📂 Project Structure

GlobalCineMindAI
├── dashboard
├── recommender
├── data
├── models
├── requirements.txt
└── README.md

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run dashboard/app.py
```
